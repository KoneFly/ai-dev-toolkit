## 底层协议

本 skill 以 **RIPER-5 协议**为骨架，该协议由 [@robotlovehuman](https://github.com/robotlovehuman) 原创设计：

- 原帖：https://forum.cursor.com/t/i-created-an-amazing-mode-called-riper-5-mode-fixes-claude-3-7-drastically/65516
- 衍生项目：[CursorRIPER](https://github.com/johnpeterman72/CursorRIPER)、[claude-code-riper-5](https://github.com/tony/claude-code-riper-5) 等

DunCanYounG-1 在此基础上叠加了嵌入式领域专用的：
- 分层架构门禁
- 四文件磁盘长任务记忆
- Git 快照回退
- Scout/Builder/Verifier 多 Agent 分工
- 比赛模式（CP-0~CP-5 检查点 + 多 Agent 并行）
